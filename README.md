# Batch Production Planner

A professional-grade production sequencing tool for optimizing multi-day, multi-line production schedules.

## Key Features
- **Cyclical Demand Planning**: Assumes demand repeats weekly, optimizing Sunday production for Monday's needs.
- **Advanced Optimization**: Uses Global Branch & Bound to guarantee the mathematical global minimum for transition time, cost, or lost sales.
- **Real-time Analytics**: Built-in dashboard with service level metrics, inventory buildup charts, and demand visualization.
- **Excel Compatibility**: Easily copy-paste demand data from Excel and export results back.
